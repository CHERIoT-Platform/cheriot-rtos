Microvium is a Javascript bytecode interpreter for embedded systems.
This port is included as a demonstration of how a high level language can be integrated and isolated in a compartment.
Please see the [javascript example](https://github.com/CHERIoT-Platform/cheriot-rtos/tree/main/examples/09.javascript/) for more information.
