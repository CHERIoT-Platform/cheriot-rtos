\brief A (mostly complete) compatibility layer for core FreeRTOS APIs.

This directory contains a (mostly compelte) compatibility layer for core FreeRTOS APIs.
Just add this directory to your include path and your code should compile without changes.
For guidance on porting code from FreeRTOS see [the programmers' guide](https://cheriot.org/book/porting_from_freertos.html).
